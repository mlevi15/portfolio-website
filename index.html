JorVi

JorVi was a website developed by Levi Magnus and Jordan Emslie that allowed users to engage in online discussions similar to forums like Reddit. The goal was to produce a similar service that allowed users to register, post content, subscribe to communities, and make comments on items. Additionally, an admin panel was created to manage users and content.

Work on the front-end required HTML5, CSS3, and JS, and the back-end was done with PHP and MySQL. My contributions included designing the UI, developing the search functionality, reporting system, and contact system, and the optimization for mobile devices.

This website was recognized as one of the best projects in the class.

View the source code on GitHub

SolaceAI

SolaceAI is a Java-based game client with a graphical user interface for the Game of the Amazons. Alongside two other team members, a Monte Carlo tree search algorithm was developed to find a winning strategy.

Our AI was put to the test in a Game of the Amazons tournament against 11 other teams that had developed their own algorithms to play the game. SolaceAI placed 3rd.

View the source code on GitHub

Analog Clock Reader

Using the Matlab Imaging Processing Toolbox and computer vision algorithms, our team of four developed a program that can read the time from images or videos of analog clocks and display it digitally.

My main contribution was the development of a custom Canny Edge Detection algorithm that performed better than the built-in function provided by the Imaging Processing Toolbox.

View the source code on GitHub

Personal Financial Platform

The goal of this project was to create a simple, user-friendly, and secure web service for managing personal finances. Our team of six used HTML5, CSS3, JS, PHP, and MySQL to build the platform.

I was one of the lead developers for this project. My major contributions included the responsive UI design, the ability for users to securely upload their bank statements to the platform, and the graphical anaylsis features that aid users in visualizing their financial information.

View the source code on GitHub
