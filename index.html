JorVi

JorVi is a website developed by Levi Magnus and Jordan Emslie that allows users to engage in online discussions similar to forums like Reddit. The goal was to produce a similar service that allows users to register, post content, subscribe to communities, and make comments on items. Additionally, an admin panel was created to manage reported content.

Work on the front-end required HTML5, CSS3, and JS, and the back-end was done with PHP and a MySQL relational database. My major contributions include optimization for mobile devices, the development of the search functionality, the reporting system, and the contact system.

This website was recognized as one of the best projects in the class.

View the source code on GitHub

Please note that this school project was originally developed in 2018 on a private repo through GitHub Classroom. The professor informed us that the repo would be made public if requested after the course completed, but our request to make the repo public was never completed. As a result, the code was uploaded to a new public repo, hence the lack of commits and proper contribution tracking.

SolaceAI

SolaceAI is a Java-based game client with a graphical user interface for the Game of the Amazons. Alongside two other team members, a Monte Carlo tree search algorithm was developed to find a winning strategy.

Our AI was put to the test in a Game of the Amazons tournament against 11 other teams that had developed their own algorithms to play the game. SolaceAI placed 3rd.

View the source code on Github

Analog Clock Reader

Using the Matlab Imaging Processing Toolbox and computer vision algorithms, our team of four developed a program that can read the time from images or videos of analog clocks and display it digitally.

My main contribution was the development of a custom Canny Edge Detection algorithm that performed better than the built-in function provided by the Imaging Processing Toolbox.

View the source code on Github

Personal Financial Platform

The goal of this project was to create a simple, user-friendly, and secure web service for managing personal finances. Our team of six used HTML5, CSS3, JS, PHP, and a MySQL relational database to build the platform.

I was one of the lead developers for this project. My major contributions include the responsive UI design, the ability for users to securely upload their bank statements to the platform, and the graphical anaylsis features that aid users in visualizing their financial information.
